很多时候我们想很方便的在windwos与Linux系统之间传互相传输文件，那么接下来我就教大家利用SecureRT工具在两个系统之间传输文件。在这里工具的下载就跳过了！

背景

有一台主机，安装了windows10，在其安装了vmware，然后安装了centos虚拟机。在windows10上安装SecureCRT来ssh连接centos虚拟机。一般在windows上面下载软件，然后通过vsftp上传到centos虚拟机上。但是这样非常麻烦，现在考虑使用sftp协议来直接传输

方案

使用SecureCRT软件ssh连接到centos虚拟机，然后在SecureCRT上面打开SFTP会话。
